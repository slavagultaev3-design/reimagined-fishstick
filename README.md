# Интерьер — Сайт мебельного магазина

## Структура проекта

```
interior-site/
├── index.html          — главная страница
├── styles/
│   └── style.css       — стили сайта
└── images/             — папка с изображениями
    ├── hero-bg.jpg     — фоновое изображение секции hero
    ├── bed-tatran.jpg  — Кровать TATRAN
    ├── chair-vilora.jpg — Кресло VILORA
    ├── table-menu.jpg  — Стол MENU
    ├── chair-askesta.jpg — Кресло ASKESTA
    ├── table-norman.jpg — Стол NORMAN
    └── sofa-nastan.jpg — Диван NASTAN
```

## Как запустить

1. Добавьте свои изображения в папку `images/` с именами, указанными выше
2. Откройте `index.html` в браузере

## GitHub Pages

1. Загрузите все файлы в репозиторий на GitHub
2. Перейдите в Settings → Pages
3. Выберите ветку `main` и корневую папку `/`
4. Сохраните — сайт будет доступен по ссылке вида `https://username.github.io/repo-name`
